API functie om een cirkel in de VGA buffer te maken van een bepaalde kleur. de dikte of opvulling hiervan is niet geïmplementeerd.

Parameters
1x, het x coördinaat van het startpunt
2y, het y coördinaat van het startpunt
3radius, de straal van de gewenste cirkel
4color, 8 bits binaire waarde van de kleur; 0brrrgggbb
5reserved, ongebruikt, kan toegepast worden voor eventuele opvulling van de cirkel. Momenteel wordt hier enkel NULL in gezet bij het aanroepen.
